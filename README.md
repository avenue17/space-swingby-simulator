# 3D Solar Swingby Simulator

태양계를 위에서 내려다보는 3D 인터페이스에서 우주선의 스윙바이 궤적을 시뮬레이션하는 객체지향프로그래밍 프로젝트입니다.

## 주요 기능

- 태양과 여러 행성의 공전 시각화
- 행성 중력권 표시
- 우주선 발사각과 초기 속도 조절
- 슬라이더 값 변경 시 궤적 실시간 재계산
- 행성 질량, 반지름, 중력권 배율 조절
- 우주선 최종 속도, 최대 속도, 최소 접근 거리 출력
- 시간에 따른 속도 그래프 표시

## 실행 방법

```bash
pip install -r requirements.txt
python app.py
